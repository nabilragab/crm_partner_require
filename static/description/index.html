CRM Partner Require

Overview

Prevent user to create opportunity without partner.

Features

  • Restrict create opportunity without choose "Customer"
  • Allow to import opportunity without customer (Useful for migration data).
  • Allow to create opportunity from incoming mail without partner.

Configuration

No configuration needed

How to Work

From pipeline, You can quick create new opportunity

From opportunity form, You must choose "Customer"

Notes:This module is tested and works on Odoo Community and Enterprise Edition.
